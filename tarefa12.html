(1)Definindo os Paradigmas: Código Aberto vs. Fechado

Um sistema operacional de código aberto, é aquele que pode ser adquirido, editado e personalizado de forma à desejar do usuário, e o sistema operacional de código fechado e aquele que é fornecido por uma empresa, não sendo possível adquirir de forma gratuita e nem edita-lo.

Vantagens e Desvantagens

Um sistema operacional de codigo aberto, tem as seguintes Vantagens: -É totalmente gratuito; -É normalmente mais leve que um sistema operacional de codigo fechado; E as seguintes Desvantagens: -É menos seguro; -Tem uma interface não amigavel;

Exemplos

Exemplos de Sistema Operacional de Código Aberto: -Ubuntu; -Fedora; -Tails; Exemplos de Sistema Operacional de Código Fechado: -Windows 10; -Windows 11; -macOS 15;
